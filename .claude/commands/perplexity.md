Use Perplexity AI to search the web for current information.

Available MCP tools (use the most appropriate one):
- `sonar_search` — Fast web search with AI summary. Best for quick lookups.
- `sonar_pro_search` — Deep search with Pro Search mode. Best for complex questions.
- `sonar_reasoning` — Analytical reasoning with web data. Best for comparisons and evaluations.
- `deep_research` — Multi-step deep research (slower, more thorough). Best for market analysis.
- `web_search` — Raw ranked URLs without AI. Best for finding specific pages. Supports batch queries with | separator.
- `structured_search` — Search with JSON schema output. Best for extracting structured data.
- `academic_search` — Scholarly sources. Best for papers and research.
- `research_job` — Specialized for freelance job research.

Tips:
- Use `recency: "week"` or `"month"` for current events
- Use `domains: "github.com,stackoverflow.com"` to focus on specific sites
- Use `search_mode: "academic"` for scholarly content
- Use `return_related: true` to get follow-up question suggestions
